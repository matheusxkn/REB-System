  1. Introdução:

    2. - Controle de gastos.

  2. Justificativa:

    3. - Tendo em vista a importância do controle financeiro e criar uma poupança em um cenário, em que a sociedade encontra-se cada vez mais consumista. A ideia de criar um controle de finanças torna-se primordial.

  3. Objetivos:
    • Gerais:

      • - Trazer para o usuário todas informações sobre gastos e reservas que houve em um determinado tempo.

    • Específicos:

      • - Manipular a informação financeira do usuário, especificar onde houve gastos excessivos. Desta forma, o usuário passa a reduzir custos.

  4. Metodologia:
    • Ferramentas Utilizadas:

      • - Linguagem de programação C# e SQL Server 2017 Configuration Manager.

      C# é uma linguagem de programação orientada a objetos, que foi desenvolvida pela Microsoft e faz parte da plataforma .NET.

      SQL Server é um sistema de gerenciador de banco de dados relacional (SGBD) desenvolvido pela microsoft.

  5. Resultados Esperados:

    6. - Obter o maior número de pessoas possíveis utilizando nossa aplicação, esperamos que ajude, porque no mundo que estamos hoje, muitas pessoas não tem controle com os gastos (sendo eles a maioria desnecessário).
